# Peaceful Tasks ✨

A minimal, calm, and delightful to-do list application with a focus on beautiful UX and smooth interactions.

## Features

- ✅ Add new tasks
- ✅ Mark tasks as completed/uncompleted
- ✅ Edit tasks inline (double-click or edit button)
- ✅ Delete tasks with smooth animations
- ✅ Persistent storage (localStorage)
- ✅ Keyboard shortcuts (Enter to add, Escape to cancel edit)
- ✅ Real-time statistics (active vs completed)
- ✅ Encouraging messages
- ✅ Beautiful empty state
- ✅ Responsive design

## Design Philosophy

### Color Palette
- **Soft, muted tones**: The app uses a calm color scheme with soft grays, gentle blues, and minimal saturation
- **Whitespace**: Generous padding and spacing create a breathing room effect
- **Subtle accents**: The purple accent color (#667eea) adds just enough personality without being overwhelming

### Interactions
- **Smooth animations**: All state changes are animated for a fluid feel
- **Micro-interactions**: Hover effects, scale transforms, and shadow transitions make the UI feel alive
- **Visual feedback**: Instant response to user actions (checkboxes, button presses, task completion)

### Typography
- **Inter font**: Clean, modern sans-serif that's highly readable
- **Weight hierarchy**: Light headings, regular body text for a gentle visual flow
- **Subtle letter spacing**: Negative letter-spacing on the title for a more elegant look

### UX Decisions
- **Task-first design**: New tasks appear at the top (most recent first)
- **Progressive disclosure**: Action buttons appear on hover to reduce visual clutter
- **Contextual encouragement**: Messages change based on progress
- **Accessible**: Proper ARIA labels and keyboard navigation

## Getting Started

Simply open `index.html` in your web browser. No build process or dependencies required!

## Keyboard Shortcuts

- **Enter**: Add a new task (when input is focused)
- **Enter**: Save edited task (when editing)
- **Escape**: Cancel editing a task

## Technical Details

- Pure vanilla JavaScript (no frameworks)
- LocalStorage for persistence
- CSS animations and transitions
- Semantic HTML5
- Responsive CSS Grid/Flexbox

## Browser Support

Works in all modern browsers that support:
- CSS Grid and Flexbox
- LocalStorage API
- ES6+ JavaScript
